Bu-kit 图标说明

请在此目录放置以下尺寸的图标文件：

1. icon_16.png  - 16x16 像素
2. icon_48.png  - 48x48 像素
3. icon_128.png - 128x128 像素

图标设计建议：
- 使用简洁的设计，便于在小尺寸下识别
- 建议使用蓝色系配色，与 Chrome DevTools 风格一致
- 可以使用锁、网络、监控等相关图形元素

临时解决方案：
如果暂时没有图标，可以使用在线工具生成：
- https://favicon.io/
- https://www.canva.com/

或者使用简单的 SVG 转 PNG 工具创建基础图标。
